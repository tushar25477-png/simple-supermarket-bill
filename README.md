# 🛒 Simple Supermarket Bill

A lightweight command-line billing application built in Python that generates bills for small retail shops. This project was developed as part of learning Python fundamentals, including functions, loops, dictionaries, lists, user input handling, and modular programming.

The application allows users to add multiple products, automatically calculates the total bill, generates a unique bill number, and prints a clean, readable invoice.

---

## 📌 Features

- ➕ Add multiple items dynamically
- 💰 Automatic price and total bill calculation
- 🧾 Generates a formatted invoice
- 🎲 Random bill number generation
- 📦 Easy-to-understand command-line interface
- ⚡ Lightweight and beginner-friendly implementation

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or later installed on your system.

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/simple-supermarket-bill.git
```

Navigate to the project directory:

```bash
cd simple-supermarket-bill
```

Run the application:

```bash
python main.py
```

---

## 📖 How It Works

1. Start the program.
2. Enter the product name.
3. Enter the quantity.
4. Enter the price per unit.
5. Repeat for additional products.
6. The application automatically:
   - Calculates each item's total.
   - Computes the grand total.
   - Generates a random bill number.
   - Displays a formatted invoice.
